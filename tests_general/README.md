# Test suite
 - Synthetic tests with Sine waves and Poiseuille flows, validating the order of accuracy of the code.
 - Comparison with respect to Ansys Fluent data for:
    - Gravity-driven flow past a square cylinder.
    - Flows past moving obstacles.
       - Steady-state tests, and transient flow with object crossing the periodic boundary.

## General information
 - An overview of the validation results can be found in the presentation: `../presentation_lbm_solver.pdf`.
 - The script `python_version_lbm_tests_general.py` can be used to generate the data found in the presentation.
 - To (further) validate the C-version of the code, each subfolder found in `test_runs` can be used as input for the C-code:
     -  If the final value of the arrays (`U, V, W, ...`) is available, the C-version of the code will automatically read the final arrays generated by Python and compare the results.
     -  The validation errors (machine precision) calculated by C are saved in the file: `report_errors.txt`.
